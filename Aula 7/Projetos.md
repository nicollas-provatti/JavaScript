# Projetos

## 1️⃣ Gerenciador de Contatos

### **Descrição:**
Desenvolva uma aplicação web simples para **armazenar, exibir, editar e excluir contatos**. Os contatos devem conter **nome, telefone e e-mail** e devem ser salvos no **LocalStorage** para que não sejam perdidos ao recarregar a página.

### **📌 Requisitos:**
- Permitir adicionar contatos através de um formulário.
- Exibir os contatos cadastrados em uma lista.
- Possuir um botão para excluir contatos individualmente.
- Utilizar o LocalStorage para armazenar os contatos.
- Implementar um campo de pesquisa para filtrar os contatos cadastrados.

### **🎯 Desafio Extra:**
- Permitir a edição dos contatos já cadastrados.
- Ordenar os contatos por ordem alfabética automaticamente.

## 2️⃣ Lista de Compras Dinâmica

### Descrição
Crie uma aplicação que permita ao usuário **adicionar itens a uma lista de compras**, removê-los e marcá-los como "comprados". Os itens devem ser salvos no **LocalStorage**, para que a lista permaneça ao recarregar a página.

### 📌 Requisitos:
- Permitir que o usuário adicione itens à lista.
- Exibir os itens adicionados em uma lista dinâmica.
- Possuir um botão para remover itens específicos.
- Utilizar **LocalStorage** para armazenar os itens.
- Criar um botão "Marcar como Comprado", que altere a aparência do item.

### 🎯 Desafio Extra:
- Implementar um contador de itens adicionados e itens comprados.
- Permitir a edição dos itens já cadastrados.

## 3️⃣ Quiz Interativo

### Descrição:
Crie um jogo de perguntas e respostas (quiz) que armazene o **desempenho do usuário no LocalStorage.** O quiz pode ter perguntas de múltipla escolha e um sistema de pontuação.

### 📌 Requisitos:
- Criar um conjunto de perguntas com múltiplas opções de resposta.
- Exibir o resultado final ao término do quiz.
- Salvar a pontuação mais alta no **LocalStorage**.
- Criar um botão "Reiniciar Quiz" para permitir que o usuário tente novamente.

### 🎯 Desafio Extra:
- Criar um temporizador para cada pergunta.
- Exibir um ranking com as pontuações mais altas registradas.

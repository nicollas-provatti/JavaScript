# Desafios de Programação em JavaScript

## Funções Básicas

1.  **Operações Aritméticas:** Crie uma função que receba dois números como entrada e exiba no console a soma, subtração, multiplicação e divisão desses números.

2.  **Classificação de Triângulos:** Elabore uma função que receba os comprimentos dos três lados de um triângulo e retorne a sua classificação: equilátero, isósceles ou escaleno.

3.  **Exponenciação:** Desenvolva uma função que calcule e retorne o resultado de um número base elevado a um expoente.

4.  **Formatação de Moeda:** Crie uma função que receba um número decimal e o formate como uma representação monetária brasileira (R$ com vírgula para centavos).

5.  **Cálculo de Juros:** Implemente duas funções: uma para calcular juros simples e outra para juros compostos, ambas recebendo capital inicial, taxa de juros e tempo como parâmetros.

6.  **Fórmula de Bhaskara:** Elabore uma função que calcule as raízes de uma equação do segundo grau utilizando a fórmula de Bhaskara. A função deve receber os coeficientes a, b e c como parâmetros e retornar um array com as raízes ou uma mensagem de erro caso o delta seja negativo.

## Lógica e Estruturas de Dados

7.  **Desempenho de Basquetebol:** Dado um array de pontuações de jogos de basquete, crie uma função que determine quantas vezes o recorde de maior pontuação foi batido e em qual jogo ocorreu a pior pontuação.

8.  **Ano Bissexto:** Escreva uma função que verifique se um determinado ano é bissexto, seguindo as regras do calendário gregoriano.

9.  **Fatorial:** Desenvolva um algoritmo que calcule o fatorial de um número inteiro.

10. **Concessionária de Veículos:** Simule um sistema de vendas de uma concessionária. Use a estrutura `switch` para determinar se a compra de um carro hatch foi bem-sucedida, oferecer outros modelos ou informar que o modelo desejado não está disponível.

11. **Aumento Salarial:** Crie uma função que calcule o aumento salarial de um funcionário com base em seu plano de trabalho (A, B ou C), onde cada plano oferece uma porcentagem de aumento diferente.

12. **Cardápio da Lanchonete:** Implemente um sistema de pedidos para uma lanchonete. Use a estrutura `switch` para calcular o valor total do pedido com base no código do item e na quantidade.  
| Código | Descrição do Produto | Preço |
|--------|----------------------|-------|
| 100    | Cachorro Quente      | R$ 3,00|
| 200    | Hambúrguer Simples   | R$ 4,00|
| 300    | Cheeseburguer        | R$ 5,50|
| 400    | Bauru                | R$ 7,50|
| 500    | Refrigerante         | R$ 3,50|
| 600    | Suco                 | R$ 2,80|

13. **Caixa Eletrônico:** Simule um caixa eletrônico que informa a quantidade mínima de notas necessárias para um determinado valor, utilizando notas de 100, 50, 10, 5 e 1 real.

14. **Plano de Saúde:** Crie uma função que calcule o valor do plano de saúde com base na idade do cliente, aplicando diferentes taxas para faixas etárias específicas.

15. **Números Ímpares em Intervalo:** Escreva uma função que imprima todos os números ímpares dentro de um intervalo definido por dois parâmetros.

16. **Anuidade de Associação:** Calcule o valor da anuidade de uma associação com juros compostos, considerando o mês de pagamento como parâmetro.

17. **Contagem em Intervalo:** Dado um array de números, conte quantos estão dentro e fora do intervalo [10, 20].

18. **Conceitos de Notas:** Converta notas numéricas em conceitos (A, B, C, D) com base em faixas de valores.

19. **Repetição de Elemento:** Crie um array que contenha um elemento repetido um determinado número de vezes.

20. **Remoção de Propriedade de Objeto:** Remova uma propriedade específica de um objeto e retorne o objeto modificado.

21. **Filtragem de Números em Array:** Crie um novo array contendo apenas os números presentes em um array de elementos diversos.

22. **Conversão de Objeto em Array de Arrays:** Transforme um objeto em um array onde cada elemento é um par chave/valor.

23. **Números Pares em Índices Pares:** Filtre um array para retornar apenas os números pares que estão em índices pares.

24. **Soma de Despesas:** Calcule o total de despesas a partir de um array de objetos, onde cada objeto representa um produto ou serviço.

25. **Jogo de Adivinhação:** Implemente um jogo onde o usuário tenta adivinhar um número aleatório gerado pelo computador.

26. **Contagem de Palavras:** Conte o número de palavras em uma string.

27. **Inversão de Chave/Valor em Objeto:** Inverta as chaves e os valores de um objeto.

28. **Melhor Estudante:** Dado um objeto com nomes de estudantes e suas notas, determine o aluno com a maior média.

29. **Filtragem de Produtos em Estoque:** Crie um array de objetos representando produtos e filtre apenas os que estão em estoque.
